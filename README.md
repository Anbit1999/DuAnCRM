# DuAnCRM

- Description: 
  Have three roles: Admin, Leader, Member
  
  Role Admin: 
      + Manage member
      + Manage group work
      + Manage user
      + Manage project
      + Login
      + Manage private account

  Role Leader:
      + Manage project
      + Manage members of project
      + Show all members
      + Login
      + Manage private account
      
  Role Member:
      + Login
      + Update processing work
      + Statistics works
      + Manage private account
